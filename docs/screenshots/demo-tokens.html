Group Access Tokens

Configure group access tokens for authenticating mirrors. Each group in your GitLab instances needs a token for HTTPS mirroring.

Note: Group access tokens are required for mirrors to authenticate via HTTPS. Create a token for each group that contains projects you want to mirror. The token must have read_repository and write_repository scopes.
